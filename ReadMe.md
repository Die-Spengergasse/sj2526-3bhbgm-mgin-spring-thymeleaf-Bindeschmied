# Instructions to use the Application
By inputing th URL http://localhost:8080 you get to the home file index.html. 
There you have two choices:
- patient list which leads you to http://localhsot:8080/patient/list and
- mob list which leads you to http://localhost:8080/mob/list  

In which you can then add a mob. After adding a mob you can input 3 parameters: 
1. Mob name its a simple String that takes any combination of characters
2. Mob Health its an int so you can only input full numbers
3. and Mob Speed which also is an int  

After clicking save you then get the options to either edit
the mob which lets you change its parameters or delete mob
which permanently deletes the mob
